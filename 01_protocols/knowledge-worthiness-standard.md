# Knowledge Worthiness Standard

## One Sentence

Worthwhile memory is not "information that happened"; it is reusable judgment that helps Codex become steadier, faster, and less error-prone.

## Capture If It Meets At Least Two

- It will likely repeat.
- It applies across threads or projects.
- It changes future action.
- It reduces future exploration.
- It prevents a repeated mistake.
- It has a clear trigger.
- It has a clear action rule.
- It has a risk, boundary, or counterexample.
- It can be named and searched briefly.

## Do Not Capture Directly

- Raw chats
- Full meeting transcripts
- Full private documents
- Media assets
- Browser profiles or cookies
- API keys or tokens
- Build outputs
- One-off temporary files

Capture the workflow, boundary, or lesson instead of the raw material.
