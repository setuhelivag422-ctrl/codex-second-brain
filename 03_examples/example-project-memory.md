# Example Project Memory

## One Sentence

This is a sanitized example of how to record a project memory without exposing private details.

## Project Boundary

- Keep project goals short.
- Record only reusable workflow knowledge.
- Do not include secrets, client data, raw files, or private links.

## Useful Memory

- Setup commands.
- Common failure modes.
- Decision rules.
- Directory conventions.
- Safe deployment steps.

## Not Useful Here

- Raw meeting notes.
- Private conversations.
- Credentials.
- Large generated outputs.
