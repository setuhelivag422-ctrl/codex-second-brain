# Codex Comfort North Star

## One Sentence

The final purpose of a second brain is to make Codex easier, calmer, and more useful to work with.

## Ask Before Adding Anything

- Will this save future explanation?
- Will this reduce repeated mistakes?
- Will this help Codex understand intent faster?
- Will this reduce friction instead of adding process?
- Will this keep the user from becoming a system maintainer?

## Rule

When complexity and comfort conflict, prefer the simpler memory that helps Codex act better.
